# Justificación técnica:
##  Explica en un párrafo por qué elegiste ese modo de red (Interna o Solo-Anfitrión) y por qué no usaste el modo "Puente" (Bridged).
Elegí el modo de red NAT porque así puedo navegar por internet sin que sea riesgoso, también descargar archivos y si alguno quiere vulnerar mis sistemas pues no podrá dañar o interferir con mi Host, después lo pienso cambiar a Red Interna ya que podría utilizar una MV (Kali ) para Atacar a mi otra MV (Windows), tambien si durante las clases llego a necesitar un archivo de mi host en mi Guest podre elegir el modo de Red Sólo Anfitrión así podre analizar archivos y mi MV no tendrá salida  a internet. No pienso utilizar el Modo de Red Bridged ya que no quiero que mis máquinas virtuales sean visibles y parezcan una computadora física (podrían ser atacadas).
